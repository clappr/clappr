How to get your stream configuration from DevTools
  1. Open your player in another tab and start playback.
  2. Open DevTools (F12) → Network tab.
  3. Stream URL: filter by .mpd — copy the Request URL of the first result (returns application/dash+xml).
  4. License Server URL: filter by your DRM domain (e.g. drm. or license) — find the POST request and copy its Request URL. If the URL contains auth params (e.g. ?deviceId=), include them.
  5. Headers: on that same POST request → Headers tab → look for custom request headers such as Authorization or X-*. Add each one below.
  6. ⚠️ MPD URLs with embedded tokens expire — grab fresh URLs each session.
Telemetry Events: Open the browser console to see telemetry events being tracked in the CONTAINER_TELEMETRY_TRACE channel.
